# 财务报告智能检索系统

基于 BM25 的轻量级财务报告检索工具，通过 LLM 查询扩展和 Parent-Child 策略，从大型 Markdown 文档中精准抽取相关章节。

## 核心特性

- **表格感知分块**：自动识别表格边界，确保表格不被切断
- **LLM 查询扩展**：自动关联相关财务报表（损益表、资产负债表、现金流量表）
- **Parent-Child 策略**：表格标签生成 + 权重增强型混合索引，提升大表格检索效果
- **Token 预算控制**：精确控制输出 Token 数量，适配 64K 上下文限制

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 设置环境变量

```bash
export QWEN_API_KEY="your-api-key"
```

### 运行检索

```bash
# 财务分析检索
source set_key.sh && python financial_retriever.py output/report.md "现金流分析"

# 通用 BM25 检索
python bm25_retriever.py output/report.md "现金流分析"
```

## Parent-Child 策略说明

**问题**：大表格内容多、关键词分散，BM25 分数偏低

**解决方案**：

| 组件 | 说明 | 作用 |
|------|------|------|
| **Child（标签）** | 纵向提取表格第一列的核心科目 | 提升宏观检索分数 |
| **Parent（完整内容）** | 完整表格内容 | 保证科目检索不漏失 |
| **混合索引** | 标签重复 3 次 + 完整表身 | 兼顾两者优势 |

**检索效果对比**：

| 检索场景 | 纯内容索引 | Parent-Child 混合索引 |
|----------|-----------|----------------------|
| 搜索"现金流量表" | ✅ 命中 | ✅ 命中（标签 TF 高） |
| 搜索"經營活動現金流量" | ✅ 命中 | ✅ 命中（标签包含） |
| 搜索"已付公司股東股息" | ✅ 命中 | ✅ 命中（表身内容保留） |
| 大表格分数 | 分散，偏低 | 集中，提升 |

## 配置参数

### BM25Retriever

```python
TABLE_TITLE_SEARCH_RANGE = 200      # 表格前搜索标题的字符范围
TABLE_TITLE_MIN_LEN = 5             # 标题最小长度
TABLE_TITLE_MAX_LEN = 30            # 标题最大长度
TABLE_TITLE_MAX_DISTANCE = 50       # 标题距离表格的最大距离
TABLE_CONTEXT_AFTER = 100           # 表格后上下文长度
PARAGRAPH_SEARCH_RANGE = 100        # 段落边界搜索范围
```

### FinancialRetriever

```python
TABLE_SCORE_MULTIPLIER = 2.0        # 表格分数权重
```

## 环境变量

| 变量名 | 说明 | 默认值 |
|--------|------|--------|
| `QWEN_API_KEY` | 通义千问 API Key | - |
| `QWEN_CHAT_URL` | Chat API URL | `https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions` |
| `QWEN_CHAT_MODEL` | 模型名称 | `qwen-plus-2025-12-01` |
| `MAX_TOKENS` | 最大输出 Token | `32768` |
